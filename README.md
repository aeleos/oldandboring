# BoringOS
A boring description for a boring os


## Features
 - Has a github repository
 - Compiles (in theory)
 - Basic VGA output
 - Heap & Paging
 - Double fault protection
 - Simple allocator with new allocator API
 - Interrupts and PIC
 - Keybaord driver with PS2 Support
